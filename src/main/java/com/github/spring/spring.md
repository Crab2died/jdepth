## IoC(控制反转, Inversion of Controller)
  1. IoC是将对象及对象之间的关系交由IoC容器管理，达到了代码关系解耦及提高了代码的可测试性
  2. 控制反转是典型的工厂模式的最佳实践
  3. 高内聚，低耦合
  4. 接口继承关系
