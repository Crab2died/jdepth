#                                            排序算法
## 一.  冒泡排序(Bubble Sort)
   1.  原理  
       数组内2相邻数据逐一比较，后一条数据大于前一条数据交换(大到小排序)，直到全部比较完成。  
       时间复杂度O(n^2) 空间复杂度O(1)
   2.  实现  
       BubbleSort

## 二.  快速排序(Fast Sort)
   1.  原理  
       快速排序是冒泡排序的改进，事先选定2个基准位置
