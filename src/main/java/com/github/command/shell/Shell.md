## 查看信息
 - `ll` or `ls -l` 等查看目录文件信息
 - `lsof -i:port` 查看端口号进程
 - `ps ef|grep 名称` 查看查找名称相关的进程信息
 - `free -g/m/b` 查看内存信息,单位G/MB/B
 - `df -h` 查看磁盘挂载情况
 - 查看端口进程 `netstat -tulnp | grep :3306`(端口号)
 - 解压 `unzip -P密码 解压包 -d 解压至目录`
 - `cat 文件 | more`   按D键翻页查看
 - `grep --color -n '.*'` 文件
